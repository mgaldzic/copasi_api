copasi API

The copasi API library wraps copasi to improve usability of the copasi library. It provides some convient functions to initialize and run simulations.

Libraries Used

  • LAPACK - implementation of routines for solving systems of simultaneous linear equations, least-squares solutions of linear systems of equations, eigenvalue problems, and singular value problems.
  • muParser - a fast math parser library
  • cURL - client-side URL transfer library needed by raptor
  • Raptor RDF Syntax Library - a set of parsers and serializers that generate Resource Description Framework (RDF) triples
  • libSBML - read, write, manipulate, translate, and validate SBML
  • Antimony - A modular human-readable, human-writeable model definition language
  • libstructural - network structural analysis based on stoichiometry matrix
  • COPASI - simulation and analysis of biochemical networks and their dynamics
